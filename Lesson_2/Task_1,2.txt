1. Дан код:
var a = 1, b = 1, c, d;
c = ++a; alert(c);           // 2
d = b++; alert(d);           // 1
c = (2+ ++a); alert(c);      // 5
d = (2+ b++); alert(d);      // 4
alert(a);                    // 3
alert(b);                    // 3
Почему код даёт именно такие результаты?

В первой строке снчало производится сложение потом присвоение значение переменной.
Во второй строке сначало происходит присвоение знчение переменной потом сложение.
В третьей строке: т.к. а уже равно 2 и в скообках сначала выполняется унарный операнд потоп выполняется бинарный операнд.
В четвертой строке выполняется то же самое, но из-за постфиксного унарного операнда переменной d присваивается значение 4.
В пятой и шестой строке выводится 3 т.к. в четвёртой и третей строке произошло инкрементирование переменных а и b унарными операндами.


2. Чему будет равен x в примере ниже?
var a = 2;
var x = 1 + (a *= 2);

x = 5 т.к. в скобках происходит умножение переменной а на 2.
